Installation Guide
==================

1. Clone the repository

$ git clone https://github.com/ioram7/cfas.git -b papai

2. Enter cfas folder

$ cd cfas

3. Create a virtual environment and activate it

$ virtualenv -p python3 env
$ source env/bin/activate

4. Install libraries into this virtual environoment

$ pip3 install django
$ pip3 install djangorestframework
$ pip3 install markdown
$ pip3 install django-filter
$ pip3 install pyeda
$ pip3 install pyparsing
$ pip3 install git+https://github.com/multiplay/mysql-connector-python

5. Update settings according to your environment (eg. DATABASE properties)

$ vi cfas/settings.py

6. Set up the database

$ python manage.py migrate
$ python manage.py createsuperuser

7. Start the server

$ env/bin/python manage.py runserver 0.0.0.0:8000
